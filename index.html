A Symbolic Notation System for Grappling

Inspired by chess notation and fighting game move lists, this project provides a living standard you can extend over time to record matches, study flows, or analyze drills.

Notation Examples

See how the notation captures complex grappling sequences in a compact, readable format.

[Gordon ↳ Andre] :: Kimura >> Americana @ Side Control

Gordon transitions from Andre, executes a Kimura that transitions to an Americana from Side Control position

[Gordon] :: Half Guard @ Guard

Gordon is in Half Guard, categorized under the Guard family of positions

[Giancarlo ↳ Ronaldo] :: Armbar @ Juji Gatame ~ Belly Down !!

Giancarlo transitions from Ronaldo, executes an Armbar at Juji Gatame position with Belly Down variation, resulting in a finish

Symbol Reference

The core symbols that make up the notation system and their meanings.

↳

Transition

Indicates a transition from one athlete to another

::

Action Separator

Separates the athlete/transition from the technique

>>

Technique Flow

Shows progression from one technique to another

@

Position Marker

Indicates the position where the technique occurs

~

Variation

Denotes a specific variation or detail of the position

!!

Finish

Marks a successful submission or match-ending sequence

[ ]

Context Brackets

Encloses athlete names and transition information
